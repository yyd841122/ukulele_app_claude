## 0. 证据分级与来源限制

### 0.1 证据标签

| 标签 | 含义 | 允许来源 |
|------|------|----------|
| ✅ Verified | 来自可信官方/用户一手观察的事实 | 官方 App Store 页面 / 官方网站 / 官方帮助与隐私页 / 用户提供的真实观察 |
| 🟡 Inferred | 基于产品类别或公开常识的合理推断 | 必须附推断依据；不得作为主证据 |
| ⚪ Unknown | 暂未通过可信来源独立确认 | 留空或在矩阵中显式标注"Unknown — 不作为依据" |

### 0.2 不允许作为主证据的来源

- 小红书 / 抖音 / 豆瓣 / Wikipedia：仅作背景了解，不入矩阵。
- Yousician / Simply Guitar / Fender Play / GuitarTuna：仅作"竞品参考"，单独成列；不作为 AI音乐学园功能的证据。
- 营销文案 / 宣传海报：不等同于产品行为；引用必须标注"Inferred (marketing claim)"。
- 反编译 APK / 抓取私有 API / 复制受版权内容：明确禁止。

### 0.3 当前 Benchmark 现状声明（重要）

> **本次 T046 任务的 Benchmark Research Agent 调研阶段被显式终止**（用户指令：不再扩大网络搜索）。本矩阵在以下约束下完成：
> - 仅纳入通过产品类别可直接观察或可信推断的功能类（如"调音器""节拍器""和弦工具"等乐理工具类作为 Verified）。
> - 任何无法独立确认的具体功能细节默认标为 ⚪ Unknown，不为填满矩阵而推断。
> - 任何后续真机/官方来源的补充都必须回填到 Verified 列，**不得**回填到 Inferred 列绕过审核。
> - 任何 v2 PRD 中引用本矩阵的"必做"项必须重新核对证据等级。

## 1. 适用范围过滤

本项目长期产品定位：**Android 平台、21 寸标准 GCEA 尤克里里、零基础到弹唱**。Benchmark 仅纳入对尤克里里有意义的功能：

| 维度 | 规则 |
|------|------|
| 平台 | 仅纳入 Android / 跨平台功能；iOS-only 排入竞品参考 |
| 乐器 | 仅纳入尤克里里相关功能；钢琴 / 吉他 / 鼓 / 古筝等排入竞品参考 |
| 网络 | 离线核心功能 vs 联网功能分开列；离线对应 v2 P1-P5，联网对应 v2 P6 |
| 内容 | 区分"原创 / 公版"vs"商业授权"；本项目严禁商业歌曲 |
