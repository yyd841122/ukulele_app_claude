## 完成标准

### T010: 构建节拍器

**功能交付物**：
- [ ] 节拍生成
- [ ] BPM 可调 (50-200)
- [ ] 节拍型选择（4/4, 3/4）
- [ ] 首拍重音

**UI 交付物**：
- [ ] MetronomePage
- [ ] BPM 调节器
- [ ] 节拍可视化
- [ ] 开始/停止控制

**验收标准**：
- [ ] 节拍按设定 BPM 发声
- [ ] 精度误差 < 5 BPM
- [ ] 声音清晰

### T011: 构建录音和回放

**功能交付物**：
- [ ] 麦克风录音（最长 5 分钟）
- [ ] 录音保存到本地
- [ ] 录音列表
- [ ] 播放控制

**UI 交付物**：
- [ ] RecordingPage
- [ ] PlaybackPage
- [ ] 录音列表

**验收标准**：
- [ ] 录音正常
- [ ] 录音可保存
- [ ] 回放正常
- [ ] 权限处理友好

### T012: 构建自评

**功能交付物**：
- [ ] 三档自评（好/一般/需改进）
- [ ] 自评与录音关联
- [ ] 自评保存

**UI 交付物**：
- [ ] SelfAssessmentDialog
- [ ] 自评结果显示

**验收标准**：
- [ ] 可选择自评等级
- [ ] 自评结果保存
- [ ] 可查看历史自评

---

## 交付物

### MetronomePage

```
路径: /metronome
功能:
- BPM 显示和调节
- 节拍型选择
- 开始/停止
- 节拍可视化
```

### RecordingPage

```
路径: /recording
功能:
- 录音按钮
- 录音时长显示
- 录音列表
```

### PlaybackPage

```
路径: /recording/:recordId
功能:
- 播放控制
- 进度显示
- 播放/暂停/停止
```

### 自评

```
功能:
- 录音后弹出自评对话框
- 选择等级
- 保存结果
```

## 风险点

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| 录音文件占用空间 | 存储问题 | 限制录音时长 5 分钟 |
| 节拍器精度不足 | 练习效果 | 技术验证 |
| 权限拒绝 | 功能不可用 | 降级提示 |
